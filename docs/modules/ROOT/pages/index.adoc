== Installation

If you want to use this extension, you need to add the `com.github.bogdanpc:quarkus-pi4j` extension first to your build file.

For instance, with Maven, add the following dependency to your POM file:

[source,xml,subs=attributes+]
----
<dependency>
    <groupId>com.github.bogdanpc</groupId>
    <artifactId>quarkus-pi4j</artifactId>
    <version>{project-version}</version>
</dependency>
----
